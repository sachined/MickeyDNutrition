# 🍔 MickeyD Nutrition Calculator

A C++ menu-driven program that calculates nutritional totals for a McDonald's meal.

## What It Does
- Displays available menu items with nutritional data
- Lets the user build a meal by selecting items
- Calculates and displays total calories, fat, protein, and carbohydrates

## Key Concepts
- Structured data in C++ (arrays / structs)
- Menu-driven user input loop
- Formatted console output
- Basic nutritional data modeling

## Build
```
bash
g++ -std=c++11 -o nutrition *.cpp
./nutrition
```
